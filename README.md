# SpecTap — Better Job Specs for Tradie Quotes

A free, open-source tool to create clear, itemised job specifications for tradespeople in under 60 seconds.

Inspired by iTabs.ai but improved with modern UI, PDF export, shareable links, and easy configuration.

## Features
- No sign-up required
- One-click copy + PDF download
- Fully configurable via JSON
- Mobile friendly
- Built with Tailwind + daisyUI

Just open `index.html` or host on GitHub Pages.

Made for Australian homeowners (especially Perth/WA).
